# MCP NumPy

MCP server exposing NumPy functionality.

## When to use this skill

Use this skill when you need to:
- Perform numerical computations
- Work with arrays and matrices
- Do linear algebra operations
- Generate random numbers
- Statistical analysis

## Tools

**Array Creation:**
- `np_array`, `np_zeros`, `np_ones`, `np_full`
- `np_arange`, `np_linspace`, `np_eye`, `np_diag`

**Array Manipulation:**
- `np_reshape`, `np_transpose`, `np_concatenate`
- `np_split`, `np_tile`, `np_repeat`
- `np_squeeze`, `np_flatten`

**Mathematical Operations:**
- `np_sum`, `np_mean`, `np_std`, `np_var`
- `np_min`, `np_max`, `np_argmin`, `np_argmax`
- `np_dot`, `np_matmul`, `np_cross`
- `np_trace`, `np_cumsum`, `np_cumprod`

**Linear Algebra:**
- `np_inv`, `np_det`, `np_eig`, `np_svd`, `np_solve`

**Random:**
- `np_rand`, `np_randn`, `np_randint`, `np_random_choice`
